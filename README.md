# Bubble rings

![Bubble rings](images/bubble-rings.gif)

## Requirements

### GLFW

Installation on Debian/Ubuntu: `sudo apt install libglfw3-dev`

### GLEW

Installation on Debian/Ubuntu: `sudo apt install libglew-dev`

## Build and execution

```
mkdir build
cd build
cmake ..
make
./bin/Tubesystem
```

for tests:

``` 
./bin/testunit
```
